# DTM plus
​
## サイト概要
### サイトテーマ
自作で作ったDesk Top Music(DTM)を投稿したり、それらを聴けるサイト
​
### テーマを選んだ理由
もともと趣味でDTMを制作していました。それを投稿するサイトは数多くあるのですが、DTM専門で投稿するサイトがなく不便だと思いました。</br>
私と同じようにDTMを中心に投稿し、それらを視聴できるサイトがあれば便利になると考え、私はこのテーマしようと思いました。</br>
### ターゲットユーザ
・DTMを制作して聞いてもらいたい人</br>
・DTMを中心に音楽を聴きたい人
​
### 主な利用シーン
・DTMを制作しそれを投稿する時</br>
・DTMを専門に音楽を視聴したい時
